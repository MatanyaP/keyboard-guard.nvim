## [0.1.0] - 2025-01-22

### Added

- Initial release
- Support for X11, Wayland, macOS, and Windows
- Layout detection for multiple environments
- Normal mode protection
- Command mode protection
- Insert mode protection (optional)
- Customizable notifications
- Debug commands
